# MeshFlow 视频稳像技术复现（离线+在线）

## 项目简介
视频稳像技术在现代视频处理领域中占据着重要地位，尤其在**无人机航拍、运动相机录制、机器人视觉导航**等实际应用场景中，有着不可替代的作用。

本人为智能科学与技术专业大二学生，对电子稳像领域抱有浓厚兴趣，在指导老师的帮助下，完成了该领域经典算法的复现工作。

本项目详细记录了刘帅成博士在 **ECCV 2016** 上发表的经典论文《**MeshFlow: Minimum Latency Online Video Stabilization**》的完整复现过程，涵盖**离线处理**与**在线处理**两种核心模式，可供相关爱好者学习参考。

## 项目包含内容
本仓库已打包整理完整的学习交流资料，具体包括：
- 离线+在线两套完整可运行代码
- 配套示例测试视频
- 调参思路与优化实战建议
- 原论文原文文件
- 代码运行演示 Demo 相关资料
